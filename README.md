## 데이터 베이스 스키마 세팅 방법

**1. create database (MySQL Client)**

```mysql
CREATE DATABASE wanted CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

**2. create tables (MySQL Client)**

```sql
< script.sql
```

- SQL 스크립트 파일은 database/script.sql 확인




## 소스코드 실행 방법

**1. Install node modules**

```shell
npm install
```

**2. build typescript**

```shell
tsc --watch
```

**3. run application**
